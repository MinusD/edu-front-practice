Технологии

Spring Framework
Универсальный фреймворк с открытым исходным кодом для Java-платформы.
ReactJS
JavaScript-библиотека с открытым исходным кодом для разработки пользовательских интерфейсов.
Tailwind CSS
Это CSS-фреймворк с открытым исходным кодом. Основная особенность этой библиотеки заключается в том, что, в отличие от других CSS-фреймворков, таких как Bootstrap, она не предоставляет ряд предопределенных классов для таких элементов, как кнопки или таблицы.
Nginx
Веб-сервер и почтовый прокси-сервер, работающий на Unix-подобных операционных системах.
Российский аналог Apache
PostgreSQL
Cвободная объектно-реляционная система управления базами данных.
Более мощная СУБД чем MySQL.
